How to use github!

Author: lhq Date: June 30, 2024  19:07:28

1.如何提交项目？

echo “# Image-Similarity-Calculation” >> README.md
git init
git add README.md
git commit -m “first commit”
git branch -M main
git remote add origin https://github.com/lhqxcc/Image-Similarity-Calculation.git
git push -u origin main

可能出现的错误:

  1. error: failed to push some refs to “github.com”

    当我们在git版本库中发现一个问题后，如你在git上对它进行了在线修改，但是没有对本地库进行同步（做到push之前，都先pull下代码，就可以保证本地库和远程库代码一致）。这个时候你再次commit，想把本地库提交到远程git库中，就会出现push失败问题。

    我们只要把远程库同步到本地库即可，使用如下命令：

    git pull –rebase origin master

    或者将commit的代码撤回，然后再git pull也行。

    git reset –soft HEAD^

  2. fatal: remote origin already exists

    先删除远程 Git 仓库(git remote rm origin),再添加远程 Git 仓库(git remote add origin+仓库地址)

  3. fatal: Couldn’t find remote ref master

    如果是新建的仓库（ repositories ）的话在pull代码的时候，出现这个提示，可以忽略不计，直接提交就可以。

2.如何在原有项目文件的基础上添加文件？

直接把本地文件拖到GitHub项目文件页面上即可
